# RPG Console ![C#](https://img.shields.io/badge/C%23-8533ff?logoSize=32)
W programie można wybrać 3 możliwe postacie: **Knight**, **Mage** i **Archer** i zrobić z nich graczy.<br>
Jest dodany widok menu i mapa, która na razie jeszcze nie jest połączona z systemem walki.<br>
- Jak skończę grę to planuję dodać zapisywanie postępu w plikach .json.

## Instalacja

### Otwórz terminal

```bash
cd (tam, gdzie chcesz przechować projekt)
git clone https://github.com/Konefka/RPG-Console.git
```

### Otwórz Eksplorator plików

```
- Przejdź do folderu RPG-Console
- Otwórz, po czym włącz RPG_Console.sln
```

> [!WARNING]
> 1. Jeżeli automatycznie nie włącza Ci się Visual studio:
>    - Naliśnij na RPG_Console.sln prawym przyciskiem - Otwórz za pomocą - Microsoft Visual Studio 2022
> 
> 2. Jeżeli wyskakuje Ci błąd, to jest szansa, że nie masz zainstalowanego .NET SDK 9.0
>    - Przejdź wtedy do https://dotnet.microsoft.com/en-us/download, zainstaluj i spróbuj jeszcze raz
